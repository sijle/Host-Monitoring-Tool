# Host-Monitoring-Tool
# 🐍 Python 全维度系统监控工具

这是一个基于 Python 和 Tkinter 开发的图形化系统资源监控工具。它能够实时显示 CPU、内存、磁盘、网络以及 NVIDIA GPU 的使用情况，并提供进程级别的资源占用列表。

## ✨ 主要功能

- **实时图表**：动态展示 CPU、GPU、内存、磁盘和网络速度的趋势图。
- **进程管理**：列出所有正在运行的进程，显示其 PID、CPU 和内存占用。
- **排序功能**：点击表格表头（如 CPU%）可以对进程进行升序/降序排列。
- **GPU 支持**：自动检测并显示 NVIDIA 显卡利用率（如果安装了 `pynvml`）。
- **自适应布局**：界面简洁，支持窗口缩放。
